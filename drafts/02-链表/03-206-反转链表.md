### 第一想法
反转链表，注意移动的顺序：

首先记下next，不然直接反转会导致找不到next；

然后把cur的next指向前一个节点pre，实现反转；

再把pre移动到cur的位置，再把cur移动到next；

顺序不能错

### 代码

```cpp
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode* pre = nullptr;
        ListNode* cur = head;

        while (cur != nullptr) {
            ListNode* next = cur->next;
            cur->next = pre;
            pre = cur;
            cur = next;
        }

        return pre;
    }
};

```
