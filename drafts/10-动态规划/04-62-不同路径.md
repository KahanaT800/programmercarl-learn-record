### 第一想法
第一个想法就是看dp[1][1]的值应该是多少，然后就想到
`dp[i][j] = dp[i - 1][j] + dp[i][j - 1]`

但是接着想一条边上的值就会发现，只要在最左和最上，不管走多远，**选择**都是1；

所以初始化的时候要初始化这两条边为1；

### 代码

```cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        vector<vector<int>> dp(m, vector<int>(n, 0));
        for (int i = 0; i < m; i++) {
            dp[i][0] = 1;
        }
        for (int i = 0; i < n; i++) {
            dp[0][i] = 1;
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
            }
        }
        return dp[m - 1][n - 1];
    }
};
```
