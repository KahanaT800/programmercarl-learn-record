### 第一想法
如何把不同顺序记录下来没想出来;

### 看完题解后的想法
组合数: 先遍历物品,再遍历容量 - 每个物品按顺序放入, 自然排除了顺序颠倒的情况

排列数: 先遍历容量, 再遍历物品 - 对于一个容量来说, 所有的物品都有可能能放进去

### 代码

```cpp
class Solution {
public:
    int combinationSum4(vector<int>& nums, int target) {
        vector<uint64_t> dp(target+1, 0);
        dp[0] = 1;

        // 排列数，先遍历容量，再遍历物品：
        for (int j = 1; j <= target; j++) {
            for (int i = 0; i < nums.size(); i++) {
                if (j >= nums[i]) {
                    dp[j] += dp[j - nums[i]];
                }
            }
        }

        return dp[target];
    }
};
```
