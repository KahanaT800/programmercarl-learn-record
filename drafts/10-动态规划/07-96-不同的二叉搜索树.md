### 看完题解后的想法
理解分开的树的序号和dp[i] 之间的关系；

卡特兰数：以 j 为根时，左子树有 j-1 个节点、右子树有 i-j 个节点，方案数相乘再对所有根求和

### 代码

```cpp
class Solution {
public:
    int numTrees(int n) {
        vector<int> dp(n+1, 0);
        dp[0] = 1;
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= i; j++) {
                dp[i] += dp[i - j] * dp[j - 1];
            }
        }
        return dp[n];
    }
};
```
