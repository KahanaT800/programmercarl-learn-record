### 第一想法
不是找完全平方数， 而是找一堆数的平方和能不能等于这个数；

注意是完全背包，因为可以复用；最少组合数和LC17一样；

dp[0] 的含义跟着题目类型走：方案数题 dp[0] = 1（空集是一种方案），最少个数题 dp[0] = 0（不需要任何元素）。

### 代码

```cpp
class Solution {
public:
    int numSquares(int n) {
        vector<int> dp(n+1, INT_MAX);
        dp[0] = 0; // 凑出 0 需要 0 个数
        
        for (int i = 0; i * i <= n; i++) {
            for (int j = i * i; j <= n; j++) {
                if (dp[j - i * i] != INT_MAX) {
                    dp[j] = min(dp[j], 1 + dp[j - i * i]);
                }
            }
        }
        return dp[n];
    }
};
```
