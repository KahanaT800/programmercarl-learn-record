### 第一想法
没啥思路， dp代表什么没想明白

### 看完题解后的想法
dp[i] 代表 i 这个数的最大拆分乘积；

然后需要遍历循环看是把这个数拆成两半的值更大，还是其中一个数还要继续拆比较大；

### 实现中遇到的困难
在状态转移的时候， 注意还需要与原来的 dp[i] 比较

### 代码

```cpp
class Solution {
public:
    int integerBreak(int n) {
        vector<int> dp(n+1, 0);
        dp[2] = 1;
        for (int i = 3; i <= n; i++) {
            for (int j = 1; j < i; j++) {
                dp[i] = max(dp[i], max((i - j) * j, dp[i - j] * j));
            }
        }

        return dp[n];
    }
};
```
