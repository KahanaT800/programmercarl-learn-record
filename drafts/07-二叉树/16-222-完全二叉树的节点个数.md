### 第一想法
层序遍历来做,但是优化方案不会;

### 看完题解后的想法
充分利用了完全二叉树的性质,如果一个棵树是完全二叉树则节点数可以直接用公式;

核心就是递归去看一个节点的左右子树高度是否一致,完全二叉树的左右子树高度一致,所以可以直接一路递归;

最后的叶子节点也会返回一个1;(左右子树都是nullptr)

### 代码

```cpp
class Solution {
public:
    int countNodes(TreeNode* root) {
        if (!root) {
            return 0;
        }
        TreeNode* leftTree = root->left;
        TreeNode* rightTree = root->right;
        int leftH = 0, rightH = 0;
        while (leftTree) {
            leftTree = leftTree->left;
            leftH++;
        }
        while (rightTree) {
            rightTree = rightTree->right;
            rightH++;
        }
        if (leftH == rightH) {
            return (2<<leftH)-1;
        }
        return countNodes(root->left) + countNodes(root->right) + 1;
    }
};
```
