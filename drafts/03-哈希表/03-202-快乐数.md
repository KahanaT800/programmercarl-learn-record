### 第一想法
没想出来如何判断无限循环；

### 看完题解后的想法
本质是判环问题。详细分析见博客：[快乐数——用 Floyd 判环算法把一道数学题变成链表题](/notes/algo/02-快乐数与floyd判环/)

核心要点：
- int 平方和上界 810，状态空间有限，必然成环
- Floyd 快慢指针判环：$O(1)$ 空间，不需要哈希表
### 实现中遇到的困难
要理解成环需要脑筋转弯一下；

### 代码

```cpp
class Solution {
public:
    int getSum(int n) {
        int sum = 0;

        while (n) {
            int d = n % 10;
            sum = sum + d * d;
            n = n / 10;
        }
        return sum;
    }

    bool isHappy(int n) {
        int slow = n;
        int fast = n;
        do {
            slow = getSum(slow);
            fast = getSum(getSum(fast));
        } while (fast != 1 && slow != fast);

        return fast == 1;
    }
};
```
