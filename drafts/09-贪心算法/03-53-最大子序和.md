### 第一想法
这道题的核心就在于如果当前值+过去值小于当前值，说明过去值可以舍去，直接用当前值作为新的起点；

分治法不会，之后研究吧；

### 实现中遇到的困难
在 `{-1}` 上踩坑了

### 代码

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int res = nums[0];
        int sum = nums[0];

        for (int i = 1; i < nums.size(); i++) {
            sum = max(nums[i], nums[i] + sum);
            res = max(res, sum);
        }
        return res;
    }
};
```
