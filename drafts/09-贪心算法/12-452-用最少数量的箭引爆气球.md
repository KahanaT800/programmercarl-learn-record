### 第一想法
不断地调整起点和终点范围，如果任意一边（其实是起点，排序之后有序）在范围内，就可以被引爆；

注意如果终点也在范围内就要调整终点位置；

```cpp
class Solution {
public:
    int findMinArrowShots(vector<vector<int>>& points) {
        sort(points.begin(), points.end());
        int sum = 1;
        int start = points[0][0];
        int end = points[0][1];
        for (int i = 1; i < points.size(); i++) {
            if (points[i][0] <= end) {
                start = points[i][0];
                if (points[i][1] <= end) {
                    end = points[i][1];
                }
            } else {
                start = points[i][0];
                end = points[i][1];
                sum++;
            }
        }

        return sum;
    }
};
```
