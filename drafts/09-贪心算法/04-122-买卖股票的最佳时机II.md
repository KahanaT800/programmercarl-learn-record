### 第一想法
可以用dp解，dp简化之后就是贪心算法；

思想就是如果遇到更低的价格就更新低价，然后继续卖出，看看价格是否比max更大

### 代码

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int price = 0;
        int minPrice = prices[0];

        for (int i = 1; i < prices.size(); i++) {
            // 如果当前股价更低，就假设在这里买入
            minPrice = min(minPrice, prices[i]);
            // 比较之后的max是否比之前的max更大
            price = max(price, prices[i] - minPrice);
        }

        return price;
    }
};
```
