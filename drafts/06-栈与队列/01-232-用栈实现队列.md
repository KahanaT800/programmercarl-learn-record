### 第一想法
一个栈 s1 用来入队，一个栈 s2 用来出队；

### 看完题解后的想法
最开始每次把数据从 s1 倒入 s2 还要把 s2 中的数据倒回去，为了保持顺序一致；

但是其实可以做一个简化，只有s2中空了才从s1中补充，这样其实是分段连续的；

### 代码

```cpp
class MyQueue {
public:
    MyQueue() {
        
    }
    
    void push(int x) {
        s1.push(x);
    }
    
    int pop() {
        transfer();
        int res = s2.top();
        s2.pop();

        return res;
    }
    
    int peek() {
        transfer();
        int res = s2.top();

        return res;
    }
    
    bool empty() {
        return s1.empty() && s2.empty();
    }
private:
    stack<int> s1;
    stack<int> s2;

    void transfer() {
        if (s2.empty()) {
            while (!s1.empty()) {
                s2.push(s1.top());
                s1.pop();
            }
        }
    }
};
```
